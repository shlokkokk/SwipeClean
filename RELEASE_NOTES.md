# Release Notes

## v1.1.0

### Added
- Home screen update button with live update status.
- GitHub auto-update support from published releases.

### Changed
- Light mode refreshed to a warmer off-white palette with a cooler accent contrast.
- Session progress now reflects the file currently being reviewed.

### Fixed
- Settings `Save Changes` only enables when there is a real net change.
- Reverting a setting back to its original value correctly disables `Save Changes` again.
- Session progress bar now reaches 100% on the final item instead of stopping at 99%.
- Release metadata now matches the tagged app version so the updater checks the correct build.
