# SwipeClean 1.0.0 - Initial Release

SwipeClean turns file cleanup into a Tinder-style swipe experience.

## Highlights
- Swipe cards to keep, delete, or skip files
- Keyboard controls: `A` (Delete), `D` (Keep), `W` (Skip)
- Smart previews for images, PDFs, and text/code files
- Session summary with files reviewed and storage saved
- Safe deletion via system trash flow

## Downloads
- Windows: `SwipeClean Setup 1.0.0.exe`

## Install (Windows)
1. Download `SwipeClean Setup 1.0.0.exe`
2. Run the installer
3. Launch SwipeClean from Start Menu

## Known Notes
- This release is unsigned, so Windows SmartScreen may show a warning. Click "More info" and then "Run anyway".
- macOS/Linux artifacts are not attached in this release from this machine.

## Troubleshooting
- If startup fails, reinstall and run as a normal user.
- If preview cache grows too large, clear it from Settings.
